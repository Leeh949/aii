# 1절 SQLite 데이터베이스 연결
- SQLite 데이터 베이스는 별도의 DBMS없이 SQL문 이용해서 DB액세스할 수 있도록 만든 간단한 디스크 기반 DB
- C라이브러리
- SQLite는 프로토타입 생성시 사용
- 프로젝트: 분석 ->  설계 ->  구현  -> 테스트  ->  고객에게 배포  ->  유지보수
            프로토타입(SQLite)     완제품(Oracle, MySQL, Maria, ...)
- DB browswe for SQLite
